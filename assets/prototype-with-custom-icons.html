Graphviz-Style Nodes with Centered Icons

Key Changes Made:

  • Icon Size: Increased from 20x20 to 40x40
  • Layout: Text now appears below icons (centered)
  • Borders: Removed all table borders
  • Spacing: Added margin between icon and text

Equivalent DOT Code:

digraph i {
    rankdir=LR;
    node [shape=none, margin=0];

    n1 [label=<
        <TABLE BORDER="0" CELLBORDER="0" CELLSPACING="0">
            <TR><TD><IMG SRC="cow.png" width="40" height="40"/></TD></TR>
            <TR><TD>some text</TD></TR>
        </TABLE>
    >];

    n2 [label=<
        <TABLE BORDER="0" CELLBORDER="0" CELLSPACING="0">
            <TR><TD BGCOLOR="lightgreen">Step</TD></TR>
            <TR><TD><IMG SRC="cow.png" width="40" height="40"/></TD></TR>
            <TR><TD>some text</TD></TR>
        </TABLE>
    >];

    n1 -> n2;
}
